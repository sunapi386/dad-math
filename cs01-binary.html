Binary: How Computers Count

A computer is billions of tiny switches, and a switch knows exactly two things: on and off. That's it. No nine, no seven, not even a three. So how does it count to a billion? The same way you'd count if someone stole all your digits except two.

⏱ ~30 min   no math lesson needed first

1 · You already use a place system

The number 342 secretly means 3\times100 + 4\times10 + 2\times1 — columns worth 100, 10, 1. Each column is ten times the one to its right, because we have ten digits.

Binary does the identical thing with two digits (0 and 1), so each column is only twice the one to its right: \dots,\ 32,\ 16,\ 8,\ 4,\ 2,\ 1.

$$\texttt{101}_2 = 1\times4 + 0\times2 + 1\times1 = 5$$

2 · Flip real bits

💡 The bit flipper

Eight switches — one byte. Click to flip. Try making your age, then 100, then the biggest number possible.

Challenges: ① make 42  ② make 255 (what do you notice?)  ③ add 1 to 255 — why can't the byte do it? (That's a real bug called overflow; old games wrapped to 0.)

3 · Count to 31 on one hand

Your fingers are switches: up = 1, down = 0. Thumb = 1, index = 2, middle = 4, ring = 8, pinky = 16. One hand counts to 1+2+4+8+16 = 31, two hands to 1023. (Watch out for the number 4.)

Why do computer numbers look so weird — 256, 1024, 65536? They're powers of two: what one more switch buys you. Every extra bit doubles what you can count.

4 · 🔮 The mind reader

Here's the superpower hiding in binary. Think of any number from 0 to 63. I'll find it in six yes/no questions — never more — because six bits is all a number that size is. Each answer you give me is exactly one bit of information.

🔮 Six questions

Think of a whole number from 0 to 63. Don't tell me. Ready?

Grown-up hint: each card is “numbers whose bit k is 1.” The yes/no answers spell the number in binary.

📓 Homework

  1. Write your birthday day-number in binary (e.g. the 23rd → 10111). Check it by adding the column values back.
  2. Finger-count from 0 to 10 in binary on one hand without looking at the page.
