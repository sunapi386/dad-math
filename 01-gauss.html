Lesson 01

The Gauss Formula

Add up every number from 1 to 100. The slow way fills half a page. The fast way fits on one line — and an eight-year-old named Carl Friedrich Gauss supposedly found it in class to annoy his teacher.

⏱ ~30 min   ages ~8+

1 · The problem

Adding a few numbers is easy:

$$1+2+3+4+5 = 15 \qquad\qquad 1+2+3 = 6$$

But what about 1 to 100? Writing it all out is painful. There has to be a shortcut — so let's build one.

2 · A shorter way to write “add them all”

Mathematicians got tired of writing 1+2+3+\cdots too. They invented the summation sign — a big Greek letter sigma, \Sigma:

$$\sum_{i=1}^{n} x_i \;=\; x_1 + x_2 + x_3 + \cdots + x_n$$
  • bottom i=1 — where the counter starts
  • top n — where it stops
  • i is the index — just a counter that walks 1, 2, 3, \dots, n. (Not “imaginary”!) It's like a loop.
  • x_i is a placeholder for each thing you add. The little i is a subscript (written low). A number written high, like x^2, is a superscript.

3 · The trick: fold it in half

Here's Gauss's idea. Take the numbers from 1 to n and pair the outside ones going in: first + last, second + second-to-last, and so on. Every pair adds up to the same total.

🤝 Pairing playground

Each purple number pairs with an orange one. Notice every pair sums to the same value — then it's just “how many pairs × that value.”

Try n = 6 by hand:   1+6 = 7,   2+5 = 7,   3+4 = 7.   Three sevens → 3 \times 7 = 21. The sum folds neatly in half between 3 and 4.

4 · Make a guess, then test it \therefore

In math you're allowed to guess — as long as you test it. We mark a guess with three dots \therefore (“therefore / hypothesis”).

Hypothesis \therefore the number of pairs is n \div 2, and each pair is worth n+1.

Test it on n = 8: pairs are 1+8, 2+7, 3+6, 4+5 — that's four pairs, each worth 9, so 4 \times 9 = 36. And 8 \div 2 = 4 ✓   8+1 = 9 ✓. The guess holds!

5 · The formula

Call the number of pairs a = \tfrac{n}{2} and the value of each pair b = n+1. The sum is a \times b:

$$\boxed{\;\sum_{i=1}^{n} i \;=\; \frac{n}{2}\,(n+1) \;=\; \frac{n(n+1)}{2}\;}$$

Check it on 1 to 100 in one line: \dfrac{100 \times 101}{2} = 5050. Done. □

6 · Detour: the multiplication table

To multiply the pairs fast, you want your times tables. Build the table and two secrets pop out: it's a mirror across the diagonal (so 3\times4 = 4\times3 — you only fill half), and numbers share common factors (3\times6 = 2\times9 = 18).

📓 Homework

  1. Fill in the multiplication table to 11 × 11 in your notebook. Use the mirror trick — fill one triangle, and you already know the other side.
  2. Use the formula (no adding one-by-one) to find 1+2+\cdots+10, \;1+2+\cdots+20, and \;1+2+\cdots+100.
  3. Challenge: the formula divides by 2. When n is odd (say 7), n/2 isn't a whole number — yet the answer still is. Why? (Hint: look at whether n or n+1 is even.)
